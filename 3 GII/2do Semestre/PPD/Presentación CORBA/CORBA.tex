\section{Definition}
\begin{frame}{¿What is CORBA?}
	\begin{block}{CORBA}
		Corba is an standard defined by Object Management Group (OMG), that allows many different components to  work together even if its' software is written in different languages or its hosted in different machines. To sum up, it makes it easy to develop distributed applications in almost any kind of situation.
	\end{block}
\end{frame}

\section{Characteristics}
\begin{frame}{Characteristics}
	\begin{itemize}
		\item Independence from the programming Language.
		\item Transparency from the distribution.
		\item Transparency from the localization.
		\item Integration from the existing software.
		\item Object activation, transparently from the client.
		\item High configuration capacity.
	\end{itemize}
\end{frame}
